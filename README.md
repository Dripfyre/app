# DripFyre - New Age Social Media Platform

A React Native app built with Expo that allows users to edit images using voice commands.

## Features

- Upload images from gallery or start with a blank canvas
- Voice-based image editing using AI (deepgram for transcription, nano-banana for image editing)
- Automatic caption and hashtag generation (gemini flash)
- Session-based editing workflow with backend sync
- Share edited images on any platform
- Catch-Up! - a new age ai first social media platform
